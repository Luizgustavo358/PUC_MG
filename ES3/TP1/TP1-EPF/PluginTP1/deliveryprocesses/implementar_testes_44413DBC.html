Steps
Selecione a técnica de implementação apropriada
Determinar a técnica apropriada para implementar o teste.
Implementar o teste
Para implementar com êxito um Script de Teste reutilizável e identificar quaisquer Solicitações de Mudança necessárias.
Recuperar o ambiente de teste para o estado conhecido
Para garantir que o ambiente seja devidamente limpo após o desenvolvimento do Script de Teste.
Resolver erros de execução
Para estabilizar o funcionamento do teste quando executado.
Restaurar o ambiente de teste para o estado conhecido
Para deixar o ambiente do jeito que você o encontrou ou no estado necessário para implementar o próximo teste.
Manter relações de rastreabilidade
Permitir que a análise de impacto e o relatório de avaliação sejam executados nos itens rastreados.
Avalie e verifique seus resultados
Para verificar se a atividade foi concluída adequadamente e se os artefatos resultantes são aceitáveis.
