Steps
Configurar o ambiente de teste para o estado conhecido
Para estabelecer com precisão o ambiente de teste em preparação para o Test Suite.
Execute o conjunto de testes
Conduzir os testes incluídos no Conjunto de Testes e monitorar e apoiar sua conclusão.
Recuperar os testes interrompidos
Para determinar a ação corretiva apropriada para recuperar-se de uma execução interrompida do Conjunto de Testes e, se necessário, corrija o problema, recupere e execute novamente o Conjunto de Testes.
Inspecione os Logs de Teste para verificar se estão completos e precisos
Para determinar se a execução do Conjunto de Testes gerou informações de teste válidas e, caso contrário, para identificar a ação corretiva apropriada.
Restaurar o ambiente de teste para o estado conhecido
Para garantir que o ambiente seja devidamente limpo após a execução do Script de Teste.
Manter relações de rastreabilidade
Permitir que a análise de impacto e o relatório de avaliação sejam executados nos itens rastreados.
Avalie e verifique seus resultados
Para verificar se a atividade foi concluída adequadamente e se os artefatos resultantes são aceitáveis.
